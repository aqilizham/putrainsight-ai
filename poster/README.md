# PutraInsight AI 2.0 A4 Poster

This folder contains the A4 portrait digital poster for the PutraInsight AI 2.0 competition prototype.

## Files

- `poster.html` - print-ready A4 poster layout.
- `poster.css` - UPM-inspired poster styling using maroon, white, gold, and subtle green.

## Open

Open `poster.html` in a browser from the project folder.

## Export

Recommended PDF export:

1. Open `poster/poster.html`.
2. Print.
3. Choose `Save as PDF`.
4. Paper size: `A4`.
5. Background graphics: `On`.
6. Margins: `None` or `Default`, depending on the browser preview.

The poster is designed for PDF or PNG export from a browser print or screenshot workflow.

## QR

The poster includes a clean QR placeholder labelled `Scan to view demo`. A generated QR image was not added because no local QR generator was available in the current environment.
