# PutraInsight AI 2.0 A4 Landscape Poster

This folder contains the redesigned A4 landscape digital poster for the PutraInsight AI 2.0 competition prototype. The layout follows the approved product showcase mockup style with a premium board composition, dashboard preview, AI insight panel, NotebookLM hub, mobile preview, and Jiwa Putra impact section.

## Files

- `poster.html` - print-ready A4 landscape poster layout.
- `poster.css` - UPM-inspired poster styling using maroon, white, gold, and subtle green.
- `assets/qr-demo.png` - generated QR code for the live demo URL.
- `PutraInsight_AI_2_0_Poster.pdf` - exported poster PDF.
- `PutraInsight_AI_2_0_Poster.png` - exported poster PNG.

## Open

Open `poster.html` in a browser from the project folder.

## Export

Recommended PDF export:

1. Open `poster/poster.html`.
2. Print.
3. Choose `Save as PDF`.
4. Paper size: `A4`.
5. Orientation: `Landscape`.
6. Background graphics: `On`.
7. Margins: `None` or `Default`, depending on the browser preview.

The poster is designed for PDF or PNG export from a browser print or screenshot workflow.
